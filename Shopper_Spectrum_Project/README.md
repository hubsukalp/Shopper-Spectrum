Project Name: Shopper Spectrum
Domain: Data Science / Machine Learning

Description:
This project performs customer segmentation using RFM analysis and KMeans clustering,
and implements an item-based collaborative filtering recommendation system.
A Streamlit application is developed for real-time customer segmentation
and product recommendation.

Contents:
- shopper_spectrum.ipynb : Complete analysis notebook
- app.py : Streamlit application
- data/ : Dataset used
- models/ : Saved ML models

How to Run:
1. Install requirements: pip install streamlit pandas numpy scikit-learn
2. Navigate to streamlit_app folder
3. Run: streamlit run app.py
